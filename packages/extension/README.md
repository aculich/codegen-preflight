# Codegen Preflight Extension

A Cursor/VS Code extension that provides developers with an up-to-date "state-of-the-world" snapshot for greenfield projects, including latest package versions and LLM model IDs.

## Features

- Daily snapshot generation from npm and PyPI
- LLM model discovery from OpenAI, Anthropic, and Gemini
- Automatic Cursor Rules integration
- MCP server with tools and resources
- Side panel UI for snapshot management

## Installation

Install from VSIX file or from the marketplace (when available).
